Set Theory Summary

1. Basic Concepts

Set: A well-defined collection of distinct objects.

  • A = {a, e, i, o, u} (roster method)
  • E = {x | x is an odd whole positive number} (descriptive method)

Finite vs. Infinite Sets:

  • Finite: D = {1, 2, 5, 9}
  • Infinite: I = {x | x is a whole number}

Notation:

  • 3 ∈ E = {1, 3, 5, ...}
  • 6 ∉ E

2. Set Relations

Equal Sets: Same elements, regardless of order.

  • {1, 2, 3, 4} = {3, 4, 1, 2}

Equivalent Sets: One-to-one correspondence.

  • {1, 2, 3} ≃ {a, b, c}

Subsets:

  • Proper Subset: B = {2, 4} ⊂ A = {1, 2, 3, 4, 5}
  • Improper Subset: A ⊆ A

Disjoint Sets:

  • {1, 3, 5} and {2, 4, 6}

3. Universal Set and Empty Set

Universal Set (U): Contains all elements under discussion.

  • U = {a, b, c, ..., z} (letters in names)

Empty Set (∅): Contains no elements.

  • Set of odd numbers divisible by 2: ∅

4. Power Set

Definition: Set of all subsets of A, denoted P(A).

  • A = {1, 2} → P(A) = {∅, {1}, {2}, {1, 2}}
  • B = {a, b, c} → P(B) = {∅, {a}, {b}, {c}, {a, b}, {a, c}, {b, c}, {a, b, c}}

5. Set Operations

Complement (A'): Elements in U but not in A.

  • U = {1, 2, 3, 4, 5}, A = {1, 3} → A' = {2, 4, 5}

Union (A ∪ B): Elements in A or B.

  • {1, 3} ∪ {2, 3} = {1, 2, 3}

Intersection (A ∩ B): Common elements.

  • {1, 2, 3} ∩ {3, 4, 5} = {3}

Difference (A - B): Elements in A but not in B.

  • {1, 2, 4} - {2, 5} = {1, 4}

6. Composite Operations

Let U = {1, 2, ..., 9}, A = {2, 3, 5, 6}, B = {3, 4, 6, 7}, C = {5, 6, 7, 8}.

  • (A ∩ B)' = {1, 2, 4, 5, 7, 8, 9}
  • A ∪ (B' ∩ C) = {2, 3, 5, 6, 8}
  • (A ∪ B)' ∩ (A ∪ C') = {1, 9}

Key Takeaways

  • Sets organize elements with shared properties.
  • Relations (equality, equivalence, subsets) define interactions between sets.
  • Operations (union, intersection, complement, difference) enable analysis and combination.
  • Power sets and composite operations handle complex scenarios.
